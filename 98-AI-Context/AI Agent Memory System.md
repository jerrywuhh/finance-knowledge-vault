# AI Agent Memory System

## 适用对象
- Codex
- Claude Code
- Cursor
- ChatGPT
- 其他 AI Agent

## 总规则
1. 不保存完整聊天记录。
2. 不保存无长期价值的临时对话。
3. 只保存长期有价值的信息。
4. 当形成经验、决策、工作流、研究框架、投资复盘、建模错误、项目推进时，更新对应文件。
