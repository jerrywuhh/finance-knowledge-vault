# Obsidian Git Plugin Guide

## 当前状态
未检测到 Obsidian Git 社区插件。

## 手动安装
1. 打开 Obsidian。
2. 进入 Settings。
3. 关闭 Restricted mode。
4. 打开 Community plugins。
5. 搜索 `Obsidian Git`。
6. 安装并启用。

## 推荐配置
- Auto commit-and-sync interval：10-30 分钟
- Auto pull interval：10-30 分钟
- Pull before push：开启
- Commit message：`vault backup: {{date}} {{time}}`
- Show status bar：开启

## 注意
在 GitHub 登录问题解决前，Obsidian Git 插件可以本地 commit，但 push 仍会失败。
