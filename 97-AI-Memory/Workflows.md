# Workflows

## Research Capture
网页 / 新闻 / 财报 / 文章 → MarkDownload → `00-Inbox/Downloaded` → [[Research Cleaner Prompt]] → `00-Inbox/Cleaned` → 正式目录。

## Research Cleaner
清理 Markdown 时必须保留全部原文，整理说明和我的理解必须与原文分开。

## Knowledge Base Audit
定期使用 [[Knowledge Base Audit Prompt]] 检查重复内容、分类冲突、异常文件名、失效链接、缺来源资料和孤立笔记。

## Topic Hub
一个文件只放一个主目录，但可以通过标签、内部链接和 Topic Hub 同时出现在多个研究主题中。
