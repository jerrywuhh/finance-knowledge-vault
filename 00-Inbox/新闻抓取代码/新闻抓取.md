
## **第一步：打开终端**

打开 Mac 的 **Terminal 终端**。

## **第二步：贴这三行**

```bash
cd "/Users/tangjingyue/Documents/Codex/2026-07-08/macos-finance-news-agent-rss-rss/finance-news-agent"
source .venv/bin/activate
python scripts/run_daily.py
```

它会自动做这些事：

```text
抓财经 RSS 新闻
筛选 AI 半导体 / 宏观 / 重点公司
生成 Markdown 日报
同步到 Obsidian
更新当天 Daily Note 链接
```

## **第三步：打开 Obsidian**

打开你的 Obsidian，进入：

```text
Finance-Knowledge
```

然后看两个地方：

```text
01-Daily-Notes/当天日期.md
02-News-Digest/当天日期-finance-news.md
```

比如今天就是：

```text
01-Daily-Notes/2026-07-08.md
02-News-Digest/2026-07-08-finance-news.md
```

## **第四步：阅读顺序**

你可以先看 Daily Note 里的链接：

```text
今日財經新聞
```

点进去看完整日报。

然后重点看这几个区块：

```text
今日最重要的 5 件事
對 AI 半導體產業鏈的影響
今日需要繼續觀察的變數
AI 半導體產業鏈
個股追蹤
```

## **第五步：自己做一句判断**

看完后，在 Daily Note 下面写一句：

```text
今日觀察：
今天最值得關注的是 ______，因為它可能影響 ______ 的估值假設。
```

比如：

```text
今日觀察：
今天最值得關注的是 Micron 記憶體價格與 AI 需求，因為它可能影響 HBM 供需與 MU 的毛利率假設。
```

所以你的早晨流程就是：

```text
打开终端
↓
贴三行指令
↓
打开 Obsidian
↓
看 Daily Note 的新闻链接
↓
写一句自己的市场判断
```

目前不需要 Codex、不需要 API，也不会额外花 token。
