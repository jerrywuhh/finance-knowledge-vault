# Research Import Guide

## 用途
所有未经整理的财经资料先进入 [[00-Inbox]]，不要直接放进正式研究目录。

## 收集范围
- 财经新闻
- 财报 PDF
- 券商报告
- 公司公告
- Earnings Call Transcript
- YouTube / CNBC / Bloomberg / WSJ 内容笔记
- 股票研究灵感
- 临时投资想法
- 面试相关素材

## 保存位置
- 原始下载内容：`00-Inbox/Downloaded`
- 附件、PDF、图片：`00-Inbox/Attachments`
- 清理后的 Markdown：`00-Inbox/Cleaned`

## 基本信息
每条资料尽量保留：标题、来源、作者、发布时间、原始链接、抓取时间、相关公司、相关概念、后续处理状态。
